# 1. Prerequisites #
* Ubuntu
* CMake
* Eigen
* OpenCV
* PCL

# 2. Running #
Clone the repository and catkin_make:
```
    git clone https://github.com/benchun123/object-based-loop-closure.git
    mkdir build
    cd build
    cmake ..
    make    
```


Launch it as follows:
```
./loop_closure_node ../data_test
```
